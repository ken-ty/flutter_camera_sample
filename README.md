# flutter_camera_sample

flutter プロジェクトでカメラを使うサンプルです。

- https://pub.dev/packages/camera/example
- https://pub.dev/packages/video_player/example

## 対応端末

- Android
  - Google Pixel 7a (実機) で動作確認済み
  - エミュレータでも動作します
- iOS
  - iPhone SE (第3世代) で動作確認済み
  - Simulator では動作しません
